# ApeFree.ApeForms

## 简介
ApeFree.ApeForms是一款基于WinForm框架的允许免费商用的UI库。ApeForms并不想重复其他知名UI库对界面美化的工作（它们已经完成的足够优秀），而是对实际开发中的常见的功能进行补充。

ApeForms项目的开发将会是一个长期的、持续的工作，会随着贡献者共同成长，不断的增加新的功能。
